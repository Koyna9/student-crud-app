# Student CRUD Application

A simple PHP and MySQL application to perform CRUD (Create, Read, Update, Delete) operations for managing student records.

## Features

- Add new students
- View all students
- Edit student information
- Delete student records

## Technologies Used

- PHP
- MySQL
- HTML/CSS
- XAMPP (Local Server)
